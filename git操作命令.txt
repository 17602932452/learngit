1、初始化git仓库
新建空文件夹，然后cd进入，之后git init ；git仓库初始化
文件夹中就会生成一个。git目录，该目录用于文件备份，被手动改。
2、提交文件到git仓库
git add 文件名 ；告诉git仓库，要提交
git commit -m "提交说明" ；提交操作
3、打印提交记录
git log；最新到最旧
4、版本回退
git reset --hard 版本号
5、查看历史命令
git reflog
6、查看仓库状态
git status;
7、查看某个文件 工作区与版本库的区别
git diff head -- readme.txt
8、撤销文件工作区的修改（还没添加到暂存区）
git checkout -- 文件名
9、撤回暂存区的内容到工作区
git reset head 文件名
10、删除文件
git rm 文件名
