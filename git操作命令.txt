1、初始化git仓库
新建空文件夹，然后cd进入，之后git init ；git仓库初始化
文件夹中就会生成一个。git目录，该目录用于文件备份，被手动改。
2、提交文件到git仓库
git add 文件名 ；告诉git仓库，要提交
git commit -m "提交说明" ；提交操作
3、打印提交记录
git log；最新到最旧
4、版本回退
git reset --hard 版本号
5、查看历史命令
git reflog
6、查看仓库状态
git status;
7、查看某个文件 工作区与版本库的区别
git diff head -- readme.txt
8、撤销文件工作区的修改（还没添加到暂存区）
git checkout -- 文件名
9、撤回暂存区的内容到工作区
git reset head 文件名
10、删除文件
git rm 文件名

远程仓库
1、注册github
2、生成ssh的公钥、私钥
ssh-keygen -t rsa -C "你的邮箱"
3、在github的settings页面添加你的ssh。
先有本地仓库，怎么与远端仓库冠梁，推送到远端仓库
1、在github上新建一个仓库。
2、与远端仓库关联
$ git remote add origin https://github.com/17602932452/learngit.git
3、第一次推送当前分支到仓库
$ git push -u origin master
4、推送更新到仓库
 git push origin master

 怎么样从远端仓库克隆一份
 git clone 远端仓库地址
